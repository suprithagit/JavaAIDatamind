DataMindCsv_AI – AI Data Analyst Dashboard (Demo Project)

An AI-powered web application that enables users to upload CSV files, query data using natural language, and receive insights through automated analysis and interactive visualizations.

Key Features
CSV file upload and in-memory data processing using Tablesaw
Natural language query support with AI-driven insights
Intelligent chart auto-selection (bar, line, pie, summary)
Interactive dashboards with real-time data statistics
Robust error handling and validation (file size, type, query limits)
Accessibility support (WCAG 2.1 compliant UI with keyboard navigation)
Tech Stack
Backend: Java 21, Spring Boot, Tablesaw, Google GenAI
Frontend: React, TypeScript, Tailwind CSS, Recharts
Architecture: REST APIs with clean, scalable design
Workflow

Users upload CSV data → system parses and analyzes it → user queries in plain English → AI interprets intent → relevant charts and insights are generated.
